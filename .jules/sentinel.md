2023-10-27 - Information Exposure via Raw Error Messages - Raw database error messages (like `err.message`) were being sent directly to the client in 500 responses. This can expose sensitive database schema details, file paths, or other internal configurations to an attacker. - Always log the detailed error server-side (e.g., using `console.error`) and return a generic error message (e.g., `{ error: "Internal server error" }`) to the client.
